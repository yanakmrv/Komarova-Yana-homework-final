# Автоматизированный мониторинг и реагирование на угрозы

## Описание проекта
Данный проект представляет собой инструмент на Python для автоматического анализа потенциальных угроз информационной безопасности. Скрипт выполняет следующие задачи:
- Сбор данных из двух источников: логов сетевой активности (Suricata) и API VirusTotal.
- Анализ IP-адресов из логов на предмет вредоносной активности.
- Имитация реагирования (блокировка подозрительных IP).
- Формирование отчёта в формате CSV.
- Визуализация результатов в виде графика (топ IP по количеству детектов).

Проект выполнен в рамках итогового домашнего задания по дисциплине «Программирование на Python».

## Используемые источники данных
1. **Логи Suricata** — файл `suricata_logs.json` в формате JSON Lines, содержащий события сетевой безопасности (поля: timestamp, src_ip, dest_ip, proto, app_proto).
2. **API VirusTotal** — публичное API для проверки репутации IP-адресов. Используется для получения статистики обнаружений (malicious, suspicious, harmless, undetected).

## Требования к окружению
- Python версии 3.8 или выше.
- Установленные библиотеки: `requests`, `pandas`, `matplotlib`.
